# 📸 Code Snap Pro

A sleek, React-powered web application that turns raw source code into beautiful, visually appealing images with syntax highlighting and aesthetic gradients. Perfect for sharing code on blogs or social media!

## ✨ Features
* **Auto-Syntax Highlighting:** Powered by Prism.js.
* **Mac-style Window Frame:** Mimics a native OS window with custom minimize/close buttons.
* **Aesthetic Gradients:** High-definition gradient backgrounds for a modern look.
* **One-Click Download:** Instantly export your customized snippet as a PNG.

## 🛠️ Tech Stack
* **Frontend:** React.js
* **Styles:** Custom CSS with CSS Flexbox
* **Libraries:** `html-to-image`, `prismjs`

## 🚀 How to Run Locally
1. Clone the repository.
2. Run `npm install` to download dependencies.
3. Run `npm start` to view it in the browser at `localhost:3000`.
